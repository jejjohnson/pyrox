# pyrox

> Probabilistic modeling with Equinox and NumPyro: Bayesian neural networks, Gaussian processes, and composable GP building blocks.

## Installation

```bash
pip install pyrox
```

Or with `uv`:

```bash
uv add pyrox
```

## Quickstart

```python
import pyrox
from pyrox import _core, gp, nn
```
